integerR Documentation

Integer Vectors

Description

as.integer coerces objects of type "integer". is.integer is a more general test of an object being interpretable as integers.

Usage

## S4 method for signature 'fm'
as.integer(x)

## S4 method for signature 'fmV'
as.integer(x)

Arguments

x

a FlashR object to be coerced or tested.

Value

as.integer returns an integer FlashR object, is.integer returns a logical value.

Examples

vec <- as.integer(fm.runif(1000, min=0, max=10))
