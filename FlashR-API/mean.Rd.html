meanR Documentation

Arithmetic Mean

Description

Compute arithmetic mean.

Usage

## S4 method for signature 'fm'
mean(x, ...)

## S4 method for signature 'fmV'
mean(x, ...)

Arguments

x

A FlashR vector or matrix.

...

further arguments passed to or from other methods.

Examples

mat <- fm.runif.matrix(100, 10)
mean(mat)
