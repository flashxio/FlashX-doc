Are some Values True?

Description

any tests whether some of the values in a set of logical vectors are true? fm.all tests whether some of the values in a single vector are true. fm.any can evaluate it lazily.

Usage

fm.any(x, lazy = FALSE)

## S4 method for signature 'fm'
any(x, ..., na.rm = FALSE)

## S4 method for signature 'fmV'
any(x, ..., na.rm = FALSE)

Arguments

x

a logical FlashR vector.

lazy

indicates whether or not to evaluate it lazily.

...

zero or more logical vectors.

na.rm

a logical indicating whether missing values should be removed.

Value

The value is a logical vector of length one.
