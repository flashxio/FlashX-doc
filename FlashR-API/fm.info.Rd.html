fm.infoR Documentation

The information of a FlashMatrix object

Description

These functions provide the basic information of a FlashMatrix object.

Usage

fm.is.sym(fm)

fm.matrix.layout(fm)

fm.is.sparse(fm)

fm.is.sink(fm)

fm.in.mem(fm)

Arguments

fm

The FlashMatrix object

Details

fm.is.sym indicates whether a matrix is symmetric.

fm.matrix.layout indicates how data in a matrix is organized. fm.is.sparse indicates whether a matrix is sparse.

fm.is.sink indicates whether a FlashMatrix object is a sink matrix.

fm.in.mem indicates whether a FlashMatrix object is stored in memory.

Value

fm.is.sym and fm.is.sparse returns boolean constants.
