printR Documentation

Print FlashMatrix objects.

Description

print prints the information of a FlashMatrix object.

Usage

## S4 method for signature 'fm'
print(x)

## S4 method for signature 'fmV'
print(x)

## S4 method for signature 'fmFactorV'
print(x)

## S4 method for signature 'fm.bo'
print(x)

Arguments

x

a FlashMatrix object. It can be a vector, a matrix or a FlashMatrix binary operator.
