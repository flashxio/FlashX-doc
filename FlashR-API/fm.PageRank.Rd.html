fm.PageRankR Documentation

Compute PageRank on a graph.

Description

This function assumes a graph is stored in a sparse matrix and computes exact PageRank with a sequence of sparse matrix multiplication.

Usage

fm.PageRank(A, d = 0.15, max.niters = 30, epsilon = 0.01,
  verbose = FALSE)

Arguments

A

a sparse matrix that stores the graph.

d

a damping factor.

max.niters

the maximal number of iterations.

epsilon

determines the precision of PageRank values.

verbose

indicates whether to print extra information.

Value

a vector that contains the PageRank value for each vertex.
